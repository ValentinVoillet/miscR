# miscR

The goal of **miscR** is to provide streamlined tools for flow cytometry data processing, including background adjustment, positivity calls using Fisher's Exact Test, and Lin's Concordance Correlation Coefficient.

## Installation

You can install the development version of `miscR` from [GitHub](https://github.com/) with:

```{r}
# install.packages("devtools")
devtools::install_github("ValentinVoillet/miscR")
```
